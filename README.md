# Booking Service — сервис бронирования переговорных комнат

REST API для автоматизации бронирования переговорных комнат в коворкинге. Сотрудники бронируют слоты, администраторы управляют комнатами и бронями. JWT-аутентификация, ролевая модель, PostgreSQL, Docker

## Требования

- Docker Desktop 4.x или выше
- Docker Compose v2
- 4 ГБ свободной оперативной памяти
- Порт 8000 (приложение) и 5432 (PostgreSQL) должны быть свободны
